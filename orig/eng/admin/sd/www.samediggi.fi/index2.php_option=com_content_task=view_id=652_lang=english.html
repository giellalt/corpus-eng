Conference room Sisti

The conference room Sisti is suitable for smaller meetings and for group works. Sisti is located on the third floor. There is a room for 15-20 people.

 

 

ROOM SIZE
* Capacity: 15-22people in the plain floor
* Area: 32 m2 (length 5,5 m, width 5,5 m)
* Only U-form: 15-20 people

TECHNOLOGY AND EQUIPMENT:
* Overhead projector
* Screen size
* Flip chart
* DVD/CD-player

* WLAN

 

CATERING SERVICES:
* Services to the conference room by order in advance

* There are no windows outside in Sisti conference room
